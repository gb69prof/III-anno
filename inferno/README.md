# Inferno di Dante — Libro vivo (PWA)

PWA didattica statica sull’Inferno di Dante.

## Contenuto
- Introduzione generale: cos’è l’Inferno, forma, terzine, endecasillabo, contrappasso, figure retoriche, struttura fisica.
- 34 canti con testo originale completo.
- Scheda per ogni canto: parafrasi ragionata, riassunto, mappa concettuale SVG, personaggi, versi chiave, peccato/pena/contrappasso.
- Ricerca interna per parole, temi e personaggi.
- Audio predisposto: inserire i file MP3 in `assets/audio/` con nome `inferno_01.mp3`, `inferno_02.mp3` ... `inferno_34.mp3`.
- Materiale allegato importato come immagini per struttura, canto I, canto III, canto V.

## Uso
Aprire `index.html` oppure caricare la cartella su GitHub Pages / Netlify / server scolastico. La modalità offline tramite service worker funziona su server HTTP/HTTPS, non aprendo direttamente il file da disco.

## Nota onesta
Il testo originale dei 34 canti è completo. Le parafrasi sono “ragionate per canto”, non ancora terzina per terzina: la struttura della PWA permette di sostituirle progressivamente con parafrasi integrali più estese.
