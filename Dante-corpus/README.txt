Dante_PWA_Opere_Complete_gbprof_v2

Versione corretta e riprogettata.

Miglioramenti principali:
- Divina Commedia divisa in cantiche e in 100 file separati, uno per canto.
- Navigatore Inferno / Purgatorio / Paradiso.
- Ricerca per parole su canti, sezioni e riassunti.
- Percorsi tematici con collegamenti interni precisi.
- Vita di Dante strutturata in fasi, con collegamenti alle opere.
- Mappa concettuale interattiva.
- Ogni opera include introduzione, temi, fase biografica, collegamenti, testo originale e guida in italiano moderno.

Nota: la sezione “italiano moderno” è una guida/parafrasi didattica, non una traduzione integrale riga per riga.

Fonti di orientamento consultate: DanteOnline, Treccani, Digital Dante (Columbia), DanteLab (Dartmouth).
I testi originali derivano dal corpus fornito dall’utente.

Uso:
Per test completo aprire con server locale o caricare su GitHub Pages / Netlify / Vercel.
Esempio locale: python3 -m http.server 8000
